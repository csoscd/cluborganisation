# ClubOrganisation (Joomla Component)

This repository contains the Joomla component **ClubOrganisation** compatible with Joomla 5 and 6.

## Linux commands to generate the project structure

```bash
mkdir -p /opt/dev/cluborganisation
cd /opt/dev/cluborganisation

# Clone or copy the repository contents into this folder
# Example:
# git clone <repo-url> .

# Ensure the Joomla component directories exist
mkdir -p administrator/components/com_cluborganisation
mkdir -p components/com_cluborganisation
```

## Iteration notes

This initial iteration delivers the component structure, database schema, administration lists/forms,
site views, and encryption handling for membership bank data. Further iterations can refine UI,
permissions, and additional UX enhancements.
